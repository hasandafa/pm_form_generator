🛠 Troubleshooting Guide

❌ "No procedures detected"

Causes:

  • Procedures not numbered (1., 2., etc.)
  • Text too short (< 10 characters)
  • Unrecognized format

Solutions:

  • Use "View Raw Data" to check structure
  • Ensure proper numbering format
  • Check for action keywords (inspect, check, etc.)

⚠️ LOV codes not generating

Causes:

  • Empty condition/action values
  • Special characters in text
  • Incorrect comma separation

Solutions:

  • Ensure values are comma-separated
  • Avoid special characters
  • Check that fields are not empty

📁 Excel file won't load

Causes:

  • Corrupted or password-protected file
  • Unsupported format
  • File permissions issue

Solutions:

  • Verify .xlsx/.xls format
  • Check file isn't password protected
  • Ensure read permissions
  • Test opening in Excel first

🔄 Duplicate identifiers error

Causes:

  • Unique identifier system malfunction
  • Corrupted tracking file

Solutions:

  • Delete unique_identifiers.json file
  • Restart application
  • Check output directory permissions
